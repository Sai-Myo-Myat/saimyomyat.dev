## Running Locally

Clone the project

```bash
git clone https://github.com/Sai-Myo-Myat/saimyomyat.dev.git
```

Go to the project directory

```bash
cd saimyomyat.dev
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn dev
```
